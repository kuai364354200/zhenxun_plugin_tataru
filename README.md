## 塔塔露现有的功能(指令)
#### 暖暖：本周时尚品鉴作业 
#### 选门：帮你选藏宝洞的门 
#### 仙人彩：帮你选每周仙人仙彩数字 
#### 看看微博：获取FF微博新闻 
#### 物品 物品名：查询物品信息 
#### 价格 大区 物品名：查询板子物价，大区不写默认豆豆柴 
#### 房子 服务器名 主城名 房子大小：查询空房。主城名为：森都、海都、沙都、白银、雪都。房子大小为：S、M、L
